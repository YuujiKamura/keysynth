These are the closest samples in the bank to the open-string set
{E2, A2, D3, G3, B3, E4} that the guitar voice unit tests exercise.
A2 / D3 / B3 are unavailable in the bank (no per-fret coverage at
those exact MIDI notes); the Bb2 / Db3 / Bb3 entries above sit one
semitone below each missing target and are used as the nearest-
neighbour reference in `tests/guitar_e2e.rs`.

These WAV files are NOT committed to the repository — they are
downloaded on demand by `tools/fetch-guitar-refs.sh` when the
`tests/guitar_e2e.rs` integration tests are run. The license file
itself IS committed so the provenance and CC0 declaration travel
with the source tree even when the WAVs aren't present.
